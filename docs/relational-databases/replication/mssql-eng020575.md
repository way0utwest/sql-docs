# MSSQL_ENG020575
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## Message Details  
  
|||  
|-|-|  
|Product Name|SQL Server|  
|Event ID|20575|  
|Event Source|MSSQLSERVER|  
|Component|[!INCLUDE[ssDEnoversion](../../includes/ssdenoversion-md.md)]|  
|Symbolic Name||  
|Message Text|Subscriber '%s' subscription to article '%s' in publication '%s' passed data validation.|  
  
## Explanation  
 The data at the Subscriber was validated against the data at the Publisher, and the data matched; therefore validation passed. For more information about validation, see [Validate Replicated Data](../../relational-databases/replication/validate-replicated-data.md).
